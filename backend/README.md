# StockMaster Backend

Backend for StockMaster inventory management system.

## Setup

1. Copy `.env.example` to `.env` and fill in your environment variables.
2. Install dependencies: `npm install`
3. Run migrations: `npx prisma migrate deploy`
4. Start server: `npm start`

## Scripts
- `scripts/db-reset.js`: Resets and seeds the local database.
- `scripts/start.sh`: Starts the backend server.
